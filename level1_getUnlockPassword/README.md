# 그루트시큐리티 모바일포렌식 #1

## 문제
>어플리케이션의 잠금 패스워드를 획득하세요.

## 조건
- 모바일 디바이스는 루팅된 상태이다.
- 어플리케이션은 잠금이 활성화 되어있다.
- 수사관은 adb를 통해 앱 데이터를 추출할 수 있다.
- 수사관은 디컴파일러를 통해 APK파일을 분석할 수 있다.

## 풀이
>모바일 디바이스에서 해당 앱 데이터를 추출할 경우 shared_prefs의 AppLockPrefs.xml에 앱 잠금 패스워드가 평문으로 작성되어있다.
